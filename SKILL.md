# Lunar MCP — Lịch Âm + BaZi MCP Server

## Mô tả
MCP Server cho lịch âm Trung Hoa: Bát Tự, chuyển đổi âm/dương lịch, Hoàng Lịch, Ngũ Hành, 24 Tiết Khí.

## Trigger Keywords
`lịch âm`, `âm lịch`, `lunar calendar`, `hoàng lịch`, `tiết khí`, `solar terms`, `ngũ hành`, `five elements`, `bazi calculation`, `农历`, `黄历`

## Setup
```bash
cd /root/.openclaw/workspace/skills/lunar-mcp
pip install lunar-python
# Hoặc dùng uv:
# uv sync
```
